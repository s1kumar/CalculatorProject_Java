# CalculatorProject_Java
This calculator works for basic functions. The user may enter any two numbers and it performs the basic functions like:
+ for Addition;
- for subtraction;
* for multiplication; 
/ for division; 
 % for modulus; 
 p for Power; 
 r for Result;
 (a+b) ^2 for a+b the whole square; 
 a2-b2 for a2-b2;
 pi for PI;
