# CalculatorProject_Java
This calculator works for basic functions. The user may enter any two numbers and it performs the basic functions like:
* '+' for Addition;
* '-' for subtraction;
* '*' for multiplication; 
* '/' for division; 
* '%' for modulus; 
* 'p' for Power; 
* 'r' for Result;
* 'ab' for (a+b)2;
* 'ab1' for a2-b2;
* 'pi' for Pi;
